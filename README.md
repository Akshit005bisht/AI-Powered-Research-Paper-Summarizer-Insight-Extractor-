# AI-Powered-Research-Paper-Summarizer-Insight-Extractor-
This repository provides an AI-driven tool to digest complex research papers and extract key insights instantly. It offers a seamless bridge between modern web interfaces and high-speed Large Language Models.

🚀 Key Features
Groq API Integration: Leverages ultra-fast inference for instant text summarization and insight extraction.

Flexible Architecture: * Monolithic (Frontend-Only): Run the application entirely through a single frontend interface using the Groq API directly.

Decoupled (Frontend-Backend): Connect a dedicated backend server to a separate frontend client for production environments.

Public Tunneling via ngrok: Expose your local development server to the internet securely using an ngrok token.

🛠️ Prerequisites
Groq API Key: Get it from Groq Console

ngrok Auth Token: Get it from ngrok Dashboard

Setup & Execution
Clone the Repo: git clone [repository-link]

Configure Keys: Add your GROQ_API_KEY and NGROK_AUTHTOKEN to the .env file.

Launch: Run the application to start summarizing papers locally or via the ngrok tunnel.
